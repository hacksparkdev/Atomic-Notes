Tags: [[Terminal]]

Open a new session

```bash
tmux new
```

#### Open new page

```bash
ctrl + b then c
```

#### Switch Between

```bash
ctrl + b 1 2 3 4 
# This is the number of the page
```

#### Show Running Sessions

```bash
tmux ls
```

#### Select a session

```bash
tmux attach 1
```

#### Split Window horizontally

```bash
Ctrl + b, then %
```

#### Split Vertically 

```bash
ctlr + b, then "
```
